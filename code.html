Autonomous Driving Toolbox

  • This Autonomous Driving toolbox is a python implementation of maximum entropy IRL for driver modeling and finding an autonomous controller for affecting the human. This is the code for RSS 2016 and IROS 2016 papers.

Controller Synthesis for Probabilistic Signal Temporal Logic Specifications (CrSPrSTL)

  • CrSPrSTL is a Matlab toolbox for Synthesizing safe controllers under Probabilistic Signal Temporal Logic (PrSTL) Specificaitons.

Diagnosis and Repair for sYnthesis (DiaRY)

  • DiaRY is a Matlab toolbox for Diagnosis and Repair of Signal Temporal Logic and Probabilistic Signal Temporal Logic formulae in a receding horizon controller synthesis problem.
